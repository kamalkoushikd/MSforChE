About Us

Learn more about the Materials Science Project.

This project was built entirely from scratch, with all components — from the backend structure to the frontend design — developed from the ground up. All the data used in the project was carefully collected and processed to ensure accuracy and completeness. The project aims to make exploring element properties more interactive, including simulating XRD patterns using Plotly.

The project focuses on presenting detailed mechanical, optical, chemical, and physical properties of elements in an intuitive format. It also integrates advanced material science data such as XRD (X-ray Diffraction) patterns for sulfur (S8) using Bragg's equation, with interactive visualizations powered by Plotly.

The project is built using Django for the backend and leverages GSAP and D3.js for frontend animations and data visualization. The data is processed using NumPy, Matplotlib, ASE, and pymatgen to provide accurate scientific insights.
